Background

I learned as an electro technician at high school. I was in love with electronics. All the chips and components just moved my mind all over the place.
After that, to be able to land a job easier, i have learned as an electrician. That was not that far from the printed circuits and code, so i reached to it.
When i finished both of this degrees and worked for like 3 years as an electro techinican / electrician, i got an irresistible offer to work as a junior programmer. I tried to create games and things like that in my freetime. When i showed one of the projects that i have made in Java to my boss, he gave me an opportunity to learn and develop as a programmer.
From that point i code all day long. Working on small and bigger projects as a self taught programmer. Managing my time on such a way to always leave enough free time for my true passion, which is coding.

Mood lamp
Video Github
It was a fun little project. This is a moodlamp. It has a web-app where you can controll it's color or turn on some animations. I have plans with this lamp in the future. I wanted to make one for each of my friends. It will turn on for everyone when someone share something on koma'k web. It has two buttons which can be used to controll the color of the lamp or turn it off/on.

Some techical details

  • It has an ESP32 microcontroller inside
  • Backend code written in C
  • Frontend uses jquery
  • Real time data flow with websockets
  • WiFi capabilities ( it can connect to a wifi network, or create it's own if there are none )
Pusher
Video Github
It is a 2D game written in JS with Jquery. Currently it's in development. It's a game where you can controll a character and move it around the screen while avoiding and killing enemies. Your player has a HP, a level and other stats. It is a local co-op game for now but i plan to make it full multiplayer and host it on my server. You and your friend can controll your characters with one keyboard. The game ends when you defeat the last boss ( The pusher ). Both player can attack each other and kill each other.

Some techical details

  • Will be hosted on my server
  • Backend code will be in node js with socket.io
  • Frontend uses jquery
